When asked to enrich test cases with test data, do the following:
- Review and understand test case thoroughly.
- Select the right test data set for the test case. Document the chosen test data set in section "Test Data" in the test case. The section "Test Data" shall be between "Description" and "Preconditions".
- For each step of the test case where the user has to enter data, mention exactly which data to enter. Add data in square brackets at the end of each step.
