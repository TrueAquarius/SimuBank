# User Story 001-003: User Logout

**As a** logged-in customer,  
**I want** to log out securely,  
**so that** my information remains protected.

**Acceptance Criteria:**
- Visible “Logout” button.
- Session invalidated immediately.
- Redirect to login with confirmation message.

**Definition of Done:**
- Logout endpoint functional.
- Session invalidation tested.
- UI feedback verified.
