# User Story 001-004: Password Reset

**As a** registered customer who forgot their password,  
**I want** to securely reset it,  
**so that** I can regain access to my account.

**Acceptance Criteria:**
- “Forgot Password” link on login page.
- Secure email reset link (expires in 15 minutes).
- New password validation and hashing.
- Success message and redirect to login.

**Definition of Done:**
- Email flow tested.
- Token security verified.
- Logging and audit checks passed.
