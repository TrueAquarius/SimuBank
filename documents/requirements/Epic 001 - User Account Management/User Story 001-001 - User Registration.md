# User Story 001-001: User Registration

**As a** new customer,  
**I want** to register for an online banking account,  
**so that** I can securely access my banking information and services.

**Acceptance Criteria:**
- Registration form with fields for name, email, mobile number, and password.
- Password validation (min 8 chars, 1 uppercase, 1 number, 1 special char).
- Email must be unique and validated.
- Successful registration redirects to login page.

**Definition of Done:**
- Code reviewed and tested.
- Passwords hashed securely.
- Registration endpoint documented.
