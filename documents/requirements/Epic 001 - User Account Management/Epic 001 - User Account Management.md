# Epic 001: User Account Management

**Epic Description:**  
As a customer of the fictitious bank, I want to be able to create and manage my online banking account securely so that I can access my financial information and perform transactions safely.

---

## Goals
- Allow users to register and create an online banking profile.
- Enable secure login/logout functionality.
- Provide password recovery/reset options.
- Ensure user data is protected and compliant with security standards.

---

## Acceptance Criteria (Epic-level)
- Users can register, log in, and log out securely.
- Passwords are hashed and never stored in plain text.
- Session timeout occurs after inactivity.
- Email verification for registration is implemented.
- All endpoints are protected and secure.
